Services

Here are the services I provide based on my experience in Web Development, System Analyst, and Software Quality Assurance.

Web Development

I develop responsive, clean, and functional websites using PHP, CodeIgniter 4, Laravel, JavaScript, HTML, CSS, and MySQL. I also design system flows and implement CRUD-based systems.

System Analysis

Experienced in gathering user requirements, creating BRD, SDD, KAK, BPMN, and UML diagrams. I analyze workflows to ensure the system meets business needs effectively.

Software Quality Assurance

I perform Functional Testing, Non-Functional Testing, Black-box Testing, Regression Testing, and Test Case design. Skilled in identifying bugs and tracking them using Jira.

UI/UX Design

Technical Documentation

I prepare structured documents such as Business Requirement Documents (BRD), Software Design Documents (SDD), and Scope of Work (KAK) using Confluence.

Database Management

Skilled in designing, optimizing, and maintaining MySQL databases, including creating ERD diagrams and ensuring efficient data structure for applications.
